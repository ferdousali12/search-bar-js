# 🔍 Real-Time Search Filter

A lightweight JavaScript project that filters a list in real time as you type — no libraries, no frameworks, just pure DOM magic.

## 🚀 Live Demo
[Click here to view](https://spectacular-kelpie-59257c.netlify.app/)

## 🛠️ Built With
- HTML
- CSS
- Vanilla JavaScript

## ✨ Features
- Instant filtering on keypress
- Smooth hover animations
- Colorful gradient list items
- Clean, minimal UI

## 📁 How It Works
Listens to the `input` event on the search box, then loops through all `<li>` items and toggles their display based on whether the text matches the query.
